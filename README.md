# repo-template

GitHub 仓库配置模板。

## 用途

新建仓库时直接用这个模板，配置一步到位，不用再手动设置 Issues、Discussions、分支保护、Merge 策略等。

## 使用



## 预配设置

| 配置项 | 值 |
|--------|-----|
| Issues | ✅ 开启 |
| Discussions | ✅ 开启 |
| Wiki | ❌ 关闭 |
| Merge 策略 | Squash + Rebase |
| 自动删分支 | ✅ 开启 |
| 分支保护 | 线性历史、禁止 force push、禁止删除、PR 流程 |
| 可见性 | 私有 |

> 此模板不放任何项目骨架代码，仅用于统一 GitHub 仓库配置。
